Strings

Strings in Python are actually a sequence, which basically means Python keeps track of every element in the string as a sequence. For example, Python understands the string "hello' to be a sequence of letters in a specific order.

String type Examples
Single word str = “Hello”
Multiple words str = “Hello world!”
Words with quotes str = ‘Hello world with “quotes”’
Print words print(“Hello World!”)

String Formatting


String formatting lets you inject items into a string rather than trying to chain items together using commas or string concatenation.

Common string formatting operators are:

  • The %s operator converts whatever it sees into a string, including integers and floats.
  • The %d operator converts numbers to integers first, without rounding.
  • The %f operator prints numbers with decimals.
    • Floating point numbers use the format %5.2f.
    • Here, 5 would be the minimum number of characters the string should contain; these may be padded with whitespace if the entire number does not have this many digits.
    • Next to this, .2f stands for how many numbers to show past the decimal point
  • The %r operator deliver the string representation of the object, including quotation marks and any escape characters.

String Example Code Output
Single Value print("I'm going to inject %s here." %'something') I'm going to inject something here.
Multiple Value print("I'm going to inject %s text here, and %s text here." %('some','more')) I'm going to inject some text here, and more text here.
Multiple passed through a variable x, y = 'some', 'more'
print("I'm going to inject %s text here, and %s text here."%(x,y)) 		I'm going to inject some text here, and more text here.
Floating point numbers print('Floating point numbers: %5.2f' %(13.144)) Floating point numbers: 13.14

Formatting with the .format() method

A better way to format objects into your strings for print statements is with the string .format() method. The syntax is:

      string here {} then also {}'.format('something1','something2')

Some of the examples of this formatting type:

Code Output
print('This is a string with an {}'.format('insert')) This is a string with an insert
print('The {2} {1} {0}'.format('fox','brown','quick')) The quick brown fox
print('First Object: {a}, Second Object: {b}, Third Object: {c}'.format(a=1,b='Two',c=12.3)) First Object: 1, Second Object: Two, Third Object: 12.3

By default, .format() aligns text to the left, numbers to the right. You can pass an optional <,^, or> to set a left, center or right alignment: 

print('{0:<8} | {1:^8} | {2:>8}'.format('Left','Center','Right'))
print('{0:<8} | {1:^8} | {2:>8}'.format(11,22,33))

#### You can precede the aligment operator with a padding character ####

print('{0:=<8} | {1:-^8} | {2:.>8}'.format('Left','Center','Right'))
print('{0:=<8} | {1:-^8} | {2:.>8}'.format(11,22,33))

Formatted String Literals (f-strings)

Introduced in Python 3.6, f-strings offer several benefits over the older .format() string method described above. For one, you can bring outside variables immediately into the string 

name = 'Fred'
print(f"He said his name is {name}.")
