Dictionary

A Python dictionary consists of a key and then an associated value. That value can be almost any Python object.

Dictionary basics


Basics Code Output
Creating Dictionary my_dict = {}
Constructing Dictionary my_dict = {
  'key1':'value1',
  'key2':'value2'
}
Create Key/value pair my_dict['key3'] = 'value3'
my_dict = {
  'key1':'value1',
  'key2':'value2',
  'key3':'value3'
}
Constructing a dictionary with multiple values my_dict = {
  'key1':'value1',
  'key2':['value21','value22'],
}

Ex1: my_dict[‘key1’]
Ex2: my_dict[‘key2’][1]




Ex1: "value1"
Ex1: "value22"
Nesting Dictionary my_dict = {
  'key1':{
    'nestkey':{
      'subnestkey':'value'
    }
  }
}

Ex: my_dict['nestkey']['subnestkey']







Ex1: "value"

Dictionary Methods


Example Method Output
my_dict = {
  'key1':'value1',
  'key2':'value2'
}
my_dict.keys()

For keys in my_dict.keys():
 print(keys) 		dict_keys(['key1', 'key2', 'key3'])
my_dict.values()

For values in my_dict.values():
  print(values) 		dict_values([1, 2, 3])
my_dict.items()

For items in my_dict.items():
  print(items) 		# Method to return tuples of all items
dict_items([('key1', 1), ('key2', 2), ('key3', 3)])
