If, elif, else Statements


if Statements in Python allows us to tell the computer to perform alternative actions based on a certain set of results.

If syntax format


if case1:
    perform action1
elif case2:
    perform action2
else:
    perform action3

If else code sample


# Example 1: If else  

  x = False
  if x:
       print('x was True!')
  else:
       print('I will be printed in any case where x is not true')

  # Output
  I will be printed in any case where x is not true

If elseif, else code sample


day = 'Saturday'

if day == 'Saturday':
    print('Today is Saturday!')
elif day == 'Sunday':
    print('Today is Sunday!')
else:
    print("It's a weekday!")

Sets

Sets are an unordered collection of unique elements. We can construct them by using the set() function.

As shown in the following example, we are going to:

  1. Initialize a set
  2. Add a value to set
  3. Show the current value of set
  4. Add the same value to set
  5. Show the current value of set


Notice how it won't place another 4 there. That's because a set is only concerned with unique elements!

Set Methods


Method Example Output
Copy set my_set = {1,2,3}

my_set_copy = my_set.copy()
my_set_copy 		{1,2,3}
Difference set my_set = {1,2,3}
my_set_diff = {1,2,3,4}

my_set_diff.difference(my_set) 		{4}
